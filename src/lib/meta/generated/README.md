# Meta Graph API - Generated Types

Este directorio contiene los tipos TypeScript generados automáticamente desde la especificación OpenAPI oficial de Meta.

## Generación de Tipos

Los tipos se generan ejecutando:

```bash
npm run openapi:generate
```

Esto descarga la especificación más reciente desde el repositorio oficial de Meta en GitHub y genera los tipos TypeScript correspondientes.

## Actualización

Para actualizar los tipos cuando Meta publique cambios:

```bash
npm run openapi:update
```

## No Editar Manualmente

⚠️ **IMPORTANTE**: Los archivos en este directorio son generados automáticamente. No edites manualmente el archivo `schema.d.ts`.
